Humos (BennuGD, GPH)

I developed this project for the GPH videogame contest.

The game was developed using BennuGD.

The idea behind it is to change the whole environment using ink and an eraser in order to create paths for the humos, some creatures that must be saved.

Each ink color has a different effect. One of them makes the humos jump, another lets them climb, walk, turn around, etc.

The game has several enemies and even the ink can be dangerous.

The technique used in this game was hard maps for collision with scroll. For drawing, the touch screen was used.

Technologies:
