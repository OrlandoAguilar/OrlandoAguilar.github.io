Android App for Arduino (Android SDK and Java)

This project was the result of my work as a teacher in a class of Advanced Robotics in the University “Universidad de Guadalajara” in Mexico.

The project consists in an application for android that controls a robot Scara by mean of a connection with an Arduino card.

To teach my class, I developed several prototypes of different parts of the application; all of them using java and the Android SDK. Some of them were:

  • Scripting with rinno
  • Animations and graphics in Android
  • Activities and services
  • Sockets and communications
  • Threads
  • Persistence, UI and more

The project uses the following technologies:

  • Sockets
  • Scripting by mean of Rhino (javascript)
  • Java
  • Android SDK
  • Threads
