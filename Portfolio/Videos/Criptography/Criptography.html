Steganography (Java and Image Processing)

This is a solo project of cryptography.

This program is able to use the Feistel algorithm to encrypt a message and hide it in an image using a key (number) to choose the channel and the bit in the image to hide the information.

The algorithm only works in the least significant bytes so that the image does not change its appearance at first sight.
