During this project, I had the opportunity to lead a group of 2 programmers.

also was in charge of researching, designing and programming the dynamics of the PUMA robot. Besides, I also designed and program part of the UI, the whole 3D graphics and physics systems and also created the some of the 3D models of the application. This project required some documentation, the SDD and the SRS. Both documents were built using the ISO standards. The project lasted 6 months. It was developed during an internship. This project was created with the following technologies:

  • QT
  • Irrlicht
  • Bullet Physics
  • CSL scripting
  • QWT
  • CSL (C Scripting Language web page of project)
  • C++
